Part II explanatory materials for Ford GoBike project

Files:
- Part_II_Explanatory.ipynb — notebook that generates three polished PNGs for slides and saves them in this `none` folder.
- Exploration_FordGoBike.ipynb — placeholder note (full notebook remains at the repository root).

How to use:
1. Open and run `Exploration_FordGoBike.ipynb` in the repository root to execute Part I (exploration).
2. Open and run `none/Part_II_Explanatory.ipynb` to generate the explanatory figures saved in `none/`.
3. Export the notebooks to HTML via Jupyter (`File > Download as > HTML`) if you need static deliverables.
